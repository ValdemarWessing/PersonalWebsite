Gesture controlled mouse

This project explores real-time hand tracking and gesture-based interaction using computer vision. Built with OpenCV and MediaPipe, it transforms hand movements captured through a webcam into direct control of the computer’s mouse and system functions.

The system detects and tracks hand landmarks in real time, distinguishing between left and right hands to assign different control roles. The right hand is used for cursor movement, where the position of the index finger is mapped to screen coordinates with smoothing applied for stable motion. The left hand controls actions such as clicking, dragging, scrolling, and triggering system shortcuts through specific gestures.

Gestures are recognized by analyzing the relative positions of finger joints and fingertips. For example, bringing the thumb and index finger together triggers a mouse click, while different finger configurations enable dragging or scrolling. A constant-power mapping is used for smooth and intuitive control, and a defined interaction area ensures precision when translating hand movement to screen space.

The project demonstrates how computer vision and real-time input processing can be used to create alternative interaction methods. It highlights an interest in human-computer interaction, gesture recognition, and designing intuitive control systems without traditional input devices.
