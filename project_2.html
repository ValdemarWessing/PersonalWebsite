CAVA-drum 2000

CAVA-drum 2000 is a digital drum synthesizer and step sequencer built in PlugData. The project explores how percussive sounds can be created entirely through real-time synthesis using oscillators, noise, filtering, and non-linear processing, without relying on prerecorded samples. The system consists of a 32-step sequencer, a collection of synthesized drum instruments, and a mixer with integrated effects. Each instrument is designed as a modular synthesis patch with controllable parameters, allowing the user to shape both the character and duration of the sound. The sequencer drives the rhythmic structure, while the mixer enables control over levels, parameter modulation, and routing to effects. Two main effects are implemented: a feedback delay network (FDN) reverb and a tempo-synchronized ping-pong delay. These add spatial depth and rhythmic variation, with parameters for feedback, level, and timing tied to the sequencer tempo. The project focuses on the relationship between technical implementation and sound design, demonstrating how relatively simple DSP techniques can be combined to create expressive and flexible musical tools.
